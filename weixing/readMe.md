#### 🏠刷题合集

1. 两数之和
  问题：
    1. 条件：一个整数数组，一个目标值
    2. 目的：找出数组中两数之和为目标值的下标，以数组形式返回
    3. 示例：
    ```
    nums = [2, 7, 11, 15], target = 9
    因为nums[0] + nums[1] = 2 + 7 = 9
    所以return [0, 1]
    ```
  分析：
      1. 问题：
          - 如何遍历数组？方法有哪些？
          - 如何向数组中添加元素？（需要返回一个下标数组）
      2. 概念：
          - 数组是什么？
          - 下标是什么？
          - js中加法的运算法则？
      3. 假设：
          - 每种输入只对应一个答案（排除返回两个及以上数组的情况，
          即数组中的数值没有重复项且每次遇到第一对满足条件的数值立即返回）
